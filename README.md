# 🎲 ARD Life – Das Spiel des Lebens (Medienedition)

**NEW RISK. NEW FUN. NEW PERSPECTIVE.**

> *Wie prägt Mediennutzung unser Denken, Vertrauen und unsere Entscheidungen?*

In **ARD Life** entwickeln wir ein spielerisches Erlebnis, das zeigt, wie sich Informationsquellen, Filterblasen und Medienkompetenz über ein ganzes Leben hinweg entwickeln – interaktiv, ehrlich und mit Mut zum Risiko.

ARD Life verbindet **Gamification und Infotainment**, um Medienkompetenz erfahrbar zu machen – nicht belehrend, sondern spielerisch, reflektiert und mit einem klaren Bezug zum öffentlich-rechtlichen Auftrag.

---

## 🎯 Das Konzept

In **ARD Life – Das Spiel des Lebens (Medienedition)** schlüpfen Spieler:innen in die Rolle von Mediennutzer:innen und durchlaufen verschiedene Lebensphasen – von der Kindheit über die Jugend bis ins Erwachsenenalter.

Jede Entscheidung beeinflusst:

- 📡 welche **Informationsquellen** genutzt werden,
- 🤝 wie sich **Vertrauen in Medien** entwickelt,
- 🫧 ob sich **Filterblasen** verstärken oder aufbrechen,
- 🔍 und wie gut **Fakten von Meinungen** unterschieden werden können.

**Zufallsereignisse** wie Fake News, gesellschaftliche Krisen oder Wahlen stellen die Spieler:innen vor neue Herausforderungen. Die Konsequenzen zeigen sich oft erst Jahre später – genau wie im echten Leben.

Am Ende steht die ehrliche Frage:

> **„Wie informiert bist du wirklich – und warum?"**

---

## 🕹️ Spielmechanik

### Spielbrett

Ein **50-Felder-Spielbrett** im Schlangenmuster (7 Spalten breit) führt die Spieler:innen durch fünf Lebensphasen:

| Phase | Felder | Meilenstein | Emoji |
|-------|--------|-------------|-------|
| Kindheit | 8 | Einschulung | 💒 |
| Jugend | 10 | Erstes Smartphone | 🧑 |
| Junges Erwachsenenalter | 12 | Erste Wahl | 🧑‍🎓 |
| Erwachsenenalter | 12 | Verantwortung | 🧑‍💼 |
| Alter | 8 | Lebenswerk | 🧓 |

### Feldtypen

- **❓ Frage** – Trivia-Frage aus ARD-Mediathek-Inhalten
- **📰 Ereignis** – Zufälliges Lebensereignis mit Entscheidung
- **⭐ Meilenstein** – Feier eines Lebensabschnitts
- **⬆️ Boost / ⬇️ Rückschlag** – Sofortige Punkteänderung
- **🎲 Chance** – Zufälliges Plus oder Minus
- **⚪ Neutral** – Durchgangsfeld

### Drei Bewertungsdimensionen

Alle Entscheidungen wirken sich auf drei Scores aus:

| Score | Emoji | Bedeutung |
|-------|-------|-----------|
| **Bildung** | 📚 | Wissen & Medienkompetenz |
| **Gemeinschaft** | 🤝 | Soziales Engagement & Zusammenhalt |
| **Glück** | ✨ | Lebenszufriedenheit & Wohlbefinden |

### Spielablauf

1. **Setup** – 1–4 Spieler:innen wählen und gestalten Avatare
2. **Drehen** – Glücksrad (1–6) bestimmt die Zugweite
3. **Landen** – Feldtyp löst Frage, Ereignis oder Meilenstein aus
4. **Entscheiden** – Antworten und Entscheidungen verändern die Scores
5. **Ergebnis** – Am Ende: Profil-Auswertung mit persönlichen Tipps

### Frageformate

- **Multiple Choice** zu ARD-Mediathek-Inhalten
- **Faktenchecks** mit Links zu Originalquellen
- **Was bedeutet das?** – Daubner Jugendwörter / Dialektwörter

### Zufallsereignisse (Random Events)

Lebensereignisse, die den Spielverlauf beeinflussen – von Fake News über Familienabende bis zu gesellschaftlichen Krisen:

| Ereignis | Auswirkung |
|----------|-----------|
| Du hast deinen Rundfunkbeitrag nicht bezahlt | −10 Gemeinschaft |
| Du liest einen Faktenfuchs-Artikel | +10 Bildung |
| Du teilst einen unüberprüften Artikel in der Familiengruppe | −5 Bildung, −5 Gemeinschaft |
| Du klickst die Überschrift, liest aber den Artikel nicht | −5 Bildung |
| Du hörst auf dem Weg zur Arbeit Deutschlandfunk | +5 Bildung |
| Du fällst auf ein Clickbait-Video rein und bist vorsichtiger | +5 Bildung |
| Du diskutierst nach einer ARD-Reportage mit Freunden | +5 Bildung, +10 Gemeinschaft |
| Du schaust die Satiresendung über Politik | +10 Glück, +5 Bildung |
| Du aktivierst Untertitel und Gebärdensprache | +5 Gemeinschaft |

### Entscheidungsfelder

Entscheidungen, die den weiteren Spielverlauf und die Fragetypen beeinflussen:

- **Welche Social-Media-Plattform benutzt du?** – Bestimmte Plattformen (Telegram, 4chan) führen zu mehr Faktencheck-Aufgaben
- **Was schaust du abends?**
- **Was hörst du am liebsten?**
- **Welche Nachrichtenform bevorzugst du?**
  - *Kurzclips (Reels/TikToks)*: +5 Unterhaltung, −5 Konzentration, +1 Extra-Faktencheck-Aufgabe
  - *Tagesschau in 100 Sekunden*: +5 Zeitgeschichte, +5 Wahrheitsscore
  - *20-Uhr-Tagesschau komplett*: +10 Zeitgeschichte, +10 Wahrheitsscore, −5 Freizeit

### KI-gestützte Fragengenerierung

Das `question_generation/`-System erzeugt automatisch Trivia-Fragen aus ARD-Mediathek-Inhalten:

1. **URLs laden** – `url_list.csv` enthält kategorisierte ARD-Mediathek-Links
2. **Gemini API** – Google Gemini liest den Seiteninhalt via URL-Context-Tool
3. **Prompt** – `prompt.md` definiert das Frageformat (Multiple Choice, 4 Optionen)
4. **Validierung** – Generierte Fragen werden gegen `question_schema.py` (Pydantic) validiert
5. **Export** – Valide Fragen werden als JSON ausgegeben und in `src/data/questions.json` integriert

### Endspiel-Profile

| Profil | Emoji | Bedingung |
|--------|-------|-----------|
| Wissens-Champion | 🎓 | Bildung ≥ 25 |
| Gemeinschafts-Held | 🤝 | Gemeinschaft ≥ 25 |
| Glückspilz | ✨ | Glück ≥ 25 |
| Allrounder | ⭐ | Alle Scores ≥ 15 |
| Entdecker | 🔭 | Auffangprofil |

## 🚀 Getting Started

### Voraussetzungen

- Node.js ≥ 18
- npm, yarn, pnpm oder bun

### Installation & Start

```bash
# Repository klonen
git clone <repository-url>
cd ard.life

# Abhängigkeiten installieren
npm install

# Entwicklungsserver starten
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000) im Browser.

### Weitere Kommandos

```bash
npm run build    # Produktions-Build
npm run start    # Produktionsserver starten
npm run lint     # ESLint ausführen
```

### Fragengenerierung (Python)

```bash
cd question_generation

# Python-Umgebung einrichten (≥ 3.13)
pip install -e .

# .env Datei erstellen mit GEMINI_API_KEY
echo "GEMINI_API_KEY=your-key-here" > .env

# Fragen generieren
python generate_question.py
```
